# Typhoon Demo Website

GitHub-Pages-fertige Band-Website für **Typhoon**.

## Dateien
- `index.html`
- `styles.css`
- `script.js`
- `typhoon-band.jpg`

## MP3-Dateien direkt ins Hauptverzeichnis hochladen
Diese Dateien bitte exakt so benennen:
- `farksilin-demo.mp3`
- `karanfil-demo.mp3`
- `gece-yine-dustun-demo.mp3`
- `bir-tek-sen-demo.mp3`
- `cilgin-demo.mp3`

## GitHub Pages
1. Neues Repository anlegen
2. Alle Dateien direkt ins Hauptverzeichnis hochladen
3. In `index.html` ersetzen:
   - `YOUR-GITHUB-USERNAME`
   - `YOUR-REPOSITORY-NAME`
4. Unter **Settings → Pages** Branch `main` / root aktivieren

## Hinweis
Die Eventeinträge sind absichtlich Demo-Inhalte zur Füllung der Website und keine bestätigten Auftritte.
